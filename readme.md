## ✨ Funcionalidades

A aplicação **Yu-Gi-Oh Jo-ken-po Edition** inclui:

1. **Música de Fundo** 🎵:
   - Trilha sonora temática para imersão no universo Yu-Gi-Oh.
2. **Animações e Estilo Temático** 💫:
   - Background animado com vídeo temático.
   - Cursor personalizado.
3. **Sistema de Jogo** 🕹️:
   - Jogadores escolhem cartas para competir contra o computador.
   - Exibição de pontuação atualizada dinamicamente (vitórias e derrotas).
4. **Design Responsivo** 📱:
   - Compatível com diversos dispositivos, mantendo a experiência de jogo.
